Synchronizing Data from Three Data Siloes

Creating an actionable report with data from three databases

August 2018 Matt Paulhus

Business Problem

How can we cross data siloes to prevent delays in our receiving inspection process?

Background

Purchased parts would arrive at receiving inspection but not have the inspection criteria set up in the Datasweep Manufacturing Execution System at the part’s current revision set in the Product Lifecycle Management system, stopping receiving and the part from continuing through the manufacturing process.

Method

I used a Java-based event script and SQL to create an actionable, proactive report for Supplier Quality Engineering. On schedule, the Agile Product Lifecycle Management system logs into the Oracle Enterprise Resource Planning system database and iterates through every item on an open purchase order. With the results, Agile retrieves the part’s Agile current revision and logs into Datasweep to queries for, using SQL, the part’s current Datasweep revision and supplier for each inspection assignment per given Datasweep inspection assigned supplier. The Agile PLM program combines the results together into a table with the following columns: Oracle part number, Datasweep supplier, Agile current revision, Datasweep revision at that particular Datasweep supplier, and the Oracle PO Promised Date ordered by Oracle PO Promised date, earliest first. The program finally emails the results to selected individuals.

Outcome

The automated report eliminates this inefficiency and is a perfect example of synchronizing information between databases to create efficiency and reduce intervention resources.

What I learned

How data siloes significantly hinder an organization's efficiency. Database Views. How blending data from multiple sources can help an organization. How simple solutions using data can help someone else's job significantly. Update in 2021: How important using a data warehouse and business intelligence application that has already blended data sources is much more efficient compared to this project.
