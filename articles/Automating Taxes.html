Automating Taxes

Why Pay When you can do it for free?

April 2021 Matt Paulhus

Problem

My wife made over 500 short-term stock trades in 2020, most of which must be itemized on Form 8949 (part of Schedule D). How can I use the IRS's Free Fillable Form online e-form to submit our taxes efficiently? With numerous stock trades, can I programmatically fill out Form 8949 and avoid paying HR Block or Turbo Tax?

Method

After consolidating all the broker statements into a single Excel file, I used a Java program to copy and paste data to the IRS e-form website.

Requirement How I Solved It
Copy and Paste from a CSV File I used a java.util.Scanner to iterate through a CSV file containing all of the security trades. I used java.awt.Robot to automate the keystrokes, namely the Ctrl+V, Ctrl+A, and the Tab keys.
Maximum 14 Rows per each Form 8949 I delineated sets of 14 rows using a Java Object[] and a javax.swing.JOptionPane to choose the correct line numbers to run. This allowed redos if needed.
Parse and Format Dates I programmed it to parse the date from the Excel file first, then format it for the e-Form.
Account for Slight Differences in Brokers Anytime the broker used the string "Various" in the date_aquired column, I used the date_sold date instead.
Slow it down I needed to have the Java program copy and paste the data fast, but not too fast that the internet browser could not keep up. I added java.lang.Thread.sleep() to slow the program down to a manageable speed.
Keep things organized I consolidated information such as the stock ticker, security type, broker name, share size, and last four of the account number for column 1A.

Outcome

It worked, and I was able to submit our taxes without external support. Cannot wait to try it again for next year's taxes!

What I learned

How easy it is to file taxes online for free and use Java to automate manual tasks.
