Predicting A Field in the PLM with 90% Accuracy

October 2021 Matt Paulhus

Business Problem

Improve the accuracy of the selected Functional Team field in the Agile Product Lifecycle Management (PLM) System.

Method

First, I needed to extract the information from the Agile PLM System using basic SQL. This resulted in a dataframe with nine dimensions and one class label. Next, I used Python to pre-process the data to create dummy variables for categorical features and remove dimensions not required for the regression model. Then, after splitting the data into predictor (X) and predicted (Y) train and test set, I ran the classifier, saving the best of 50 iterations.

Step 1: Develop the original SQL using Oracle SQL Developer

Step 2: Pre-process the data. I needed to replace the placeholder product line numeric values with list values and create the dummy variables for select columns.

Step 3: Conduct iterations of training data and save the best classification.

Step 4: Assess the results. Use the Classification Report to assess class labels with low precision and recall scores and consider:

  • Can we remove the class labels (list values in the PLM) with low usage to remove clutter and maintenance costs?
  • Can we improve how to use class labels with low F1 scores?
  • Were any false positives originally mislabeled?
  • Can we improve the PLM system so that, as users, we can improve how accurately we choose the Functional Team field?

    • Future Outcome

    Deploy one program that periodically updates the classification model saved within the PLM system. Use the Agile PLM System's internal event scripting to automatically fill in the Functional Team field with a predicted value. The classification model will periodically retrain using changes that we, the Configuration Management Analysts, make. Also, I can use this framework for more advanced models, such as predicting correct reviewers for product and document changes.

    What I learned

    Machine Learning Basics, Regression and Classification, Python, Confusion Matrix, scoring a Regression or Classification Model (using accuracy, precision, recall, and F1 score), statistics, and how to save a classification and use it to predict new values.
